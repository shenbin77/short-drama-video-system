## 文档维护规则

- 每个主题尽量只有一份权威文档
- 根 `README.md` 只做首页导航，不再承载整套百科说明
- 运行方式、入口、断点续跑规则统一收口到 `CURRENT_ENTRYPOINTS_RUNBOOK.md`
- 阶段性大文档保留，但必须明确标记为历史/参考，不再与权威文档竞争

详细维护规则见：

- `DOCUMENTATION_RULES.md`
