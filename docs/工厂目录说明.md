# services 输出目录说明

主程序路径保持英文，避免脚本路径和第三方工具编码问题。
人工查看用中文子目录：

- novel-factory/output/正式章节：小说正文成品
- novel-factory/queue/待分镜章节：给分镜工厂消费的章节队列
- storyboard-factory/output/分镜图片：每集分镜图与 episode.json
- video-factory/output/视频片段：Seedance/Dreamina 生成的视频片段
- assembly-factory/output/成品视频：最终可发布成片
- assembly-factory/temp/临时合成：FFmpeg/TTS 临时文件

生产主线：小说工厂  分镜工厂  视频工厂  合成工厂。
